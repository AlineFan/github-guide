How Does GBrain Work?

From your notes to AI-powered answers

Your Data Journey

Step 1: Import Your Files
Drop your Markdown notes, emails, or calendar data into GBrain. Example: you import 10,000+ files including meeting notes, contact profiles, and project ideas.
Step 2: Smart Indexing
GBrain splits your files into chunks and creates vector embeddings — so it understands meaning, not just keywords.
Step 3: Hybrid Search
When you ask a question, GBrain uses both keyword matching and semantic understanding. Search "last meeting with the client" and it finds relevant notes even without the word "client."
Step 4: AI Answers with Context
Your AI assistant now has full context. Ask "Prepare for next week's meeting with John" and it knows who John is, what you discussed last time, and what follow-ups were promised.
Step 5: Dream Cycle (While You Sleep)
Overnight, GBrain automatically organizes the day's conversations — filling gaps, fixing references, merging duplicate info. You wake up to a smarter AI.

Quick Start

Three steps, under 5 minutes

Step 1: Install GBrain

TERMINAL # Clone the repo and install git clone https://github.com/garrytan/gbrain.git ~/gbrain && cd ~/gbrain curl -fsSL https://bun.sh/install | bash export PATH="$HOME/.bun/bin:$PATH" bun install && bun link gbrain --version # Verify installation

Step 2: Initialize your brain

TERMINAL export OPENAI_API_KEY=sk-... # Your OpenAI API key gbrain init # Create local database (2 seconds) gbrain doctor --json # Check everything is OK

Step 3: Import and search

TERMINAL gbrain import ~/your-notes/ # Import your Markdown notes gbrain embed --stale # Generate search index gbrain query "What were my key themes this month?"
🎉
Done!

Your AI now has long-term memory. Try asking: "Summarize my recent notes" or "Who did I last discuss investment with?"
